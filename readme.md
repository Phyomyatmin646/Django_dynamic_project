## About

This is my personal portfolio website built to showcase my:

- Projects and completed works
- Professional skills and expertise
- Resume and education
- Work experience
- Contact information

It is designed to be responsive, clean, and easy to navigate.

---

## Features

- Dynamic project portfolio section with categories (e.g., Wordpress, Development, Design)
- Interactive skills progress bars
- Resume and education sections
- Admin panel for managing content dynamically
- Contact form to reach out to me
- Smooth animations and modern UI design

---

## Technologies Used

- **Backend:** Django 6.0
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap/Tailwind CSS
- **Database:** SQLite (default) / can be configured with PostgreSQL
- **Rich Text Editor:** CKEditor for dynamic content
- **Version Control:** Git
